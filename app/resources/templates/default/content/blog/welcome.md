This is your first blog post. You can edit this file or add new `.md` files to the `content/blog/` directory.

## How it works

Each markdown file in `content/blog/` becomes a blog post. Add frontmatter at the top for the title, date, and excerpt:

```
---
title: "My Post Title"
date: "2026-03-01"
excerpt: "A short description of the post."
---
```

The rest of the file is the post content, written in markdown.
