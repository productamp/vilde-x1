# Phase 3 ŌĆö Scaffold Starter

## Goal

When a user creates a new project, the app copies a bundled Vite+React+Tailwind+shadcn template into the project directory. No blank page, no manual setup.

## Stack

Every ProductVibe project uses the same stack:

- Vite 7 (build tool)
- React 19 (UI)
- TypeScript (language)
- Tailwind CSS 4 (styling, via `@tailwindcss/vite` plugin)
- shadcn/ui (55 components, Nova style, base-nova)
- Lucide (icons)
- React Router (routing) ŌĆö TODO: not yet in template
- Inter (font, via `@fontsource-variable/inter`)
- Recharts (charts)

## Template location

```
app/resources/templates/default/
```

Bundled into the Electron app. On new project creation, copy these files to the user's project directory, then run `npm install`.

## How the template was created

```bash
# 1. Scaffold
npx shadcn@latest create --preset "https://ui.shadcn.com/init?base=base&style=nova&baseColor=neutral&theme=neutral&iconLibrary=lucide&font=inter&menuAccent=subtle&menuColor=default&radius=default&template=vite&rtl=false" --template vite

# 2. Add all components
npx shadcn@latest add --all
```

### Preset breakdown

| Parameter | Value | Notes |
|-----------|-------|-------|
| base | base | Base layer |
| style | nova | shadcn Nova style |
| baseColor | neutral | Neutral color palette |
| theme | neutral | Neutral theme |
| iconLibrary | lucide | Lucide icons |
| font | inter | Inter font family |
| menuAccent | subtle | Subtle menu accents |
| menuColor | default | Default menu colors |
| radius | default | Default border radius |
| template | vite | Vite template (not Next.js) |
| rtl | false | Left-to-right |

## What's in the template

- `package.json` ŌĆö Vite+React+shadcn deps, scripts: `dev`, `build`, `lint`, `preview`
- `components.json` ŌĆö shadcn config (base-nova style, path aliases)
- `vite.config.ts` ŌĆö React plugin + Tailwind plugin, `@` alias
- `src/components/ui/` ŌĆö 55 shadcn components (all installed)
- `src/lib/utils.ts` ŌĆö `cn()` helper
- `src/hooks/` ŌĆö shadcn hooks
- `src/index.css` ŌĆö Tailwind + shadcn theme variables
- `src/App.tsx` ŌĆö entry component
- `src/main.tsx` ŌĆö React root
- `index.html` ŌĆö HTML entry point

## Approach ŌĆö Option B (bundled template)

Decision: **Bundle the template inside the app.** Faster, works offline, no dependency on `npx` at runtime.

Flow:
1. User creates new project ŌåÆ app copies `resources/templates/default/` to project directory
2. App runs `npm install` in the project directory
3. Project is ready for Phase 4 (Vite dev server)
