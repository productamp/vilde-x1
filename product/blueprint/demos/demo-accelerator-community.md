# Demo — Accelerator / Startup Community

## Audience

Accelerator teams, startup communities, and program operators who already have a website but need faster updates, better design quality, and consistent content output.

## Demo story

This demo shows one clear narrative:

1. **Clone existing website into ProductVibe**
   - Use website migration with a live URL.
   - ProductVibe captures the current site and converts it into an editable React project.

2. **Redesign inside ProductVibe**
   - Improve visual quality using design controls, blocks, and designer-quality output.
   - Show a modernized homepage quickly without rebuilding from scratch.

3. **Generate content (blog default)**
   - Use Content/CMS workflow (with blog as default) to generate new posts aligned to the accelerator's voice and audience.
   - Show how the same product handles both site design and ongoing content production.

## Why this demo works

- Starts from a real existing website (high credibility).
- Shows clear before/after transformation (high visual impact).
- Ends with ongoing content capability, not just a one-time redesign.

## Success criteria

- Existing site imported successfully.
- At least one redesigned page shown.
- At least one content item (blog post by default) generated and editable.
- Audience understands this is a repeatable workflow, not a one-off stunt.
