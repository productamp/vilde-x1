# Feature — Designer Quality Output

Add a feature focused on improving visual quality of generated outputs.

This is a high-impact feature:
- Use designer-focused Claude Code skills to improve output quality.
- Embed this quality approach into the master prompt.
- Add a repeatable process to polish designs toward impeccable style quality.
- Reference approach: https://impeccable.style/

## User value

- Better visual quality from the very first generation.
- Less generic AI design output.
- More production-ready, design-forward results without manual design expertise.

## Behavior

- Works on top of normal generation flow (same chat workflow).
- Uses Claude Code as the engine with upgraded design skill instructions.
- Applies design-quality pass/polish process before final output.
- Can be used for both initial generation and iterative refinements.

## Process direction

- Start from user request.
- Generate first pass.
- Run design polish pass (typography, spacing, hierarchy, visual rhythm, motion, consistency).
- Return improved version as the default output quality bar.

## Goal

Raise baseline design quality to a premium standard and keep outputs competitive with best-in-class AI website builders.
