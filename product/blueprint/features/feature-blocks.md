# Feature — Blocks

Add a blocks feature in edit mode.

This is a simple product feature:
- In edit mode, user clicks the `+` add button.
- A block picker dialog opens.
- User chooses a block and inserts it directly into the page.
- This is mainly for visually exciting demos (especially YouTube content).

## User flow

1. User opens edit mode.
2. User clicks `+`.
3. Block picker dialog appears.
4. User selects a block (for example: hero, testimonials, CTA, gallery).
5. Block is inserted into the page.

## Highlight block

- Include a premium/fancy hero block option.
- Example reference: Adeline-style hero with a zoom-out effect.
- This should be easy to apply for high-impact visual demos.

## Behavior

- Block insertion should be fast and no-code.
- Inserted blocks are fully editable after adding.
- Works with existing chat/edit workflow (chat can continue refining the result).
- Designed to increase “wow” moments in demos without extra complexity.

## Goal

Make it very easy to add high-quality visual sections quickly, so users can create exciting pages with minimal effort.
